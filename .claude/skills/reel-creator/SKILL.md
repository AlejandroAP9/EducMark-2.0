# Reel Creator — Reels Profesionales Sin Camara

> No necesitas mostrar tu cara. Necesitas mostrar tu PRODUCTO y tu MENSAJE.

---

## Formatos de Reel Sin Camara

### 1. Screen Recording + Texto Superpuesto
- Grabar pantalla de EducMark en accion
- Texto aparece sincronizado con lo que pasa
- Ideal para: demos, tutorials rapidos
- Duracion: 15-30 seg

### 2. Texto Cinetico (Kinetic Text)
- Fondo solido (negro, azul oscuro, beige)
- Texto aparece frase por frase
- Ideal para: contrarian, frases de impacto, estadisticas
- Duracion: 15-30 seg

### 3. POV / Solo Manos
- Camara apunta a escritorio/manos, NUNCA a la cara
- Ideal para: pila de pruebas, usar celular (OMR)
- Duracion: 15-45 seg

### 4. Before/After Split Screen
- Pantalla dividida: izquierda (antes) / derecha (despues)
- Ideal para: comparar proceso manual vs EducMark
- Duracion: 15-30 seg

### 5. Slideshow Animado
- Screenshots/imagenes con transiciones
- Texto superpuesto narrando
- Ideal para: resultados, paso a paso
- Duracion: 15-45 seg

---

## Estructura Temporal

**Reel 30 seg:**
```
0:00-0:03  HOOK (texto grande)
0:03-0:05  Transicion/setup visual
0:05-0:20  CUERPO
0:20-0:25  RESOLUCION
0:25-0:30  CTA
```

**Reel 15 seg:**
```
0:00-0:02  HOOK
0:02-0:12  CUERPO
0:12-0:15  CTA
```

## Elementos de Produccion

### Musica
- Demos: tech/moderna suave
- Contrarian: silencio o sonido dramatico sutil
- Emocional: piano suave, acustica
- NUNCA musica que compita con el texto

### Texto en Pantalla
- Font: bold, sans-serif, legible en movil
- Tamano: GRANDE (media atencion del profe)
- Posicion: centro o tercio superior
- Enfasis: palabra clave en color diferente

### Transiciones
- Corte seco (comun y dinamico)
- Zoom in/out (enfatizar)
- Swipe (antes/despues)
- Fade (solo para cierre)
