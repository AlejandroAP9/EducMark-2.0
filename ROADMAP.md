# ROADMAP — EducMark

> Max 12 items. Solo P0s. Revision cada lunes.
> Inspirado en Anthropic: "Monthly planning, max 12 things, just P0s."
> Cada item tiene 1 DRI (responsable) y estado binario: on track o no.

## Estrategia de Lanzamiento: Beta Pioneros

> "The way you really lose trust is if you release it early and then nothing ever happens."
> — Jenny Wen, Design Lead @ Anthropic

EducMark se lanza como **Beta para Profesores Pioneros**, no como producto terminado.
El compromiso publico: cada semana se mejora algo visible basado en feedback real.
Esto valida el Plan Pionero anual — no es un descuento, es una promesa de co-creacion.

**Regla:** No esperar a que este perfecto. Lanzar, iterar, mostrar velocidad. La confianza se construye con velocidad de respuesta, no con perfeccion inicial.

## Reglas del ROADMAP

1. **Max 12 items** — Si quieres agregar uno, quita otro primero
2. **Solo P0s** — Si no es critico para este mes, no va aqui
3. **Revision semanal** — Cada lunes con el monday-kickoff
4. **Estado honesto** — "On track" o "Bloqueado" con razon
5. **No features especulativas** — Solo cosas respaldadas por feedback o estrategia

## Revision Semanal

1. Cada P0: on track o bloqueado?
2. Algun P0 ya no es P0? → Mover a backlog

4. Que aprendi esta semana que cambia prioridades?
